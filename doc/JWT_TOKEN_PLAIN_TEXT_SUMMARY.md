## Token Format

### Example JWT Token (Plain Text)
```
eyJhbGciOiJIUzI1NiJ9.eyJpZCI6ImExYjJjM2Q0LWU1ZjYtNzg5MC1hYmNkLWVmMTIzNDU2Nzg5MCIsImV4cGlyZXNBdCI6MTczNTg2MjQwMDAwMH0.Xh7j9K2mN4pQ6rS8tU0vW1xY2zA3bC4dE5fF6gG7hH8
```

### Character Set
- **Allowed characters:** `A-Z`, `a-z`, `0-9`, `-`, `_`, `.`
- **No spaces, no newlines, no special characters**
- **Portable across all platforms and systems**
- **Length:** Approximately 150-200 characters

### Structure
```
[Header].[Payload].[Signature]
   ↓        ↓          ↓
Base64  Base64    Base64
```

### Method 1: API Call (cURL)

```bash
# Generate token
curl -k -X POST "https://your-domain.com/api/auth/token/generate?userId=USER_ID_HERE" \
  -H "Content-Type: application/json" \
  -d '{"expirationMinutes": 43200}'

# Response (JSON with plain text token)
{
  "success": true,
  "message": "Token generated successfully",
  "data": {
    "token": "eyJhbGciOiJIUzI1NiJ9.eyJpZCI6ImExYjJjM2Q0...",
    "expirationMinutes": "43200"
  }
}
```

**The token is in `data.token` as a plain string**


## Implementation Details

### Backend (Java/Netty)

**JwtService.java** - Token Generation
```java
String jwt = Jwts.builder()
    .claim("id", tokenId)
    .claim("expiresAt", expiration.toEpochMilli())
    .signWith(signingKey)
    .compact();  // ← Returns plain text string

return new TokenPair(jwt, tokenId, expiration);
```

**RerouteHandler.java** - API Response
```java
userService.generateToken(userId, expirationMinutes)
    .subscribe(jwt -> {  // ← jwt is a String
        Map<String, String> data = new HashMap<>();
        data.put("token", jwt);  // ← Plain text string in JSON
        sendJsonResponse(ctx, req, OK, ApiResponse.success(..., data));
    });
```

## Token Usage in Client

### Python Example

```python
# Load token from file
with open('illiad_jwt_token.txt', 'r') as f:
    jwt_token = f.read().strip()  # Plain text string

# Build Illiad header
def build_jwt_header(jwt_token):
    crypto_type = 0x07  # JWT
    token_bytes = jwt_token.encode('utf-8')

    # Length field
    length = 2 + 1 + len(token_bytes)
    header = struct.pack('>H', length)

    # Crypto type
    header += bytes([crypto_type])

    # JWT token (plain text → bytes)
    header += token_bytes

    # CRLF immediately after token (no offset/padding in strict mode)
    header += b"\r\n"

    return header

# Send to server
import ssl, socket
sock = socket.socket()
tls_sock = ssl.wrap_socket(sock)
tls_sock.connect(('your-domain.com', 443))
tls_sock.send(build_jwt_header(jwt_token))
```

### JavaScript Example

```javascript
// Token from user input
const jwtToken = document.getElementById('token-input').value;

// Convert to bytes
const encoder = new TextEncoder();
const tokenBytes = encoder.encode(jwtToken);

// Build header
const header = new Uint8Array([
    // Length (2 bytes, big-endian)
    ((tokenBytes.length + 3) >> 8) & 0xFF,
    (tokenBytes.length + 3) & 0xFF,
    // Crypto type
    0x07,
    // Token bytes
    ...tokenBytes,
    // CRLF immediately after token
    0x0D, 0x0A
]);

// Send via WebSocket or fetch API
socket.send(header);
```

## Security Notes

### Token is Plain Text BUT:

1. **Transmitted over TLS/SSL** - Encrypted in transit
2. **Signed with HMAC** - Cannot be forged or modified
3. **Contains no sensitive data** - Only UUID and expiration
4. **Validated server-side** - Database lookup required
5. **Revocable** - Can be invalidated immediately
6. **Time-limited** - Expires after chosen duration

### Best Practices

✅ **DO:**
- Store in secure location (password manager)
- Use HTTPS for all API calls
- Rotate tokens regularly (30 days recommended)
- Download and delete from server UI

❌ **DON'T:**
- Share tokens with others
- Post in public forums
- Commit to version control
- Email without encryption
